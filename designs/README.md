# Design Assets

Figma designs for implementation reference.

## Structure

```
designs/
├── full/        # Full page screenshots
└── sections/    # Individual section screenshots
```

## Naming Convention

- `full/home.png` - Home page full design
- `full/product-list.png` - Product list page
- `full/product-detail.png` - Product detail page
- `sections/header.png` - Header section
- `sections/footer.png` - Footer section
- `sections/hero.png` - Hero section
- `sections/product-card.png` - Product card component
